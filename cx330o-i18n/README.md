# cx330o i18n — 多语言 + 多币种国际化

i18next 生态集成，覆盖前端国际化全场景。

## 功能

- 50+ 语言支持（内置主流语言包）
- React 组件级翻译（react-i18next Hook API）
- 多币种格式化（135+ 货币）
- RTL 布局支持（阿拉伯语 / 希伯来语）
- AI 辅助翻译（LLM 保持上下文一致性）
- 翻译文件热更新

## 安装

```bash
npm install i18next react-i18next i18next-http-backend i18next-browser-languagedetector
```

## 语言包结构

```
cx330o-i18n/locales/
├── en/translation.json
├── zh/translation.json
├── ja/translation.json
├── es/translation.json
├── ar/translation.json
├── de/translation.json
├── fr/translation.json
└── pt/translation.json
```
