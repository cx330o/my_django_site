# cx330o DocForge

PDF invoice and quote generation microservice for the cx330o Sales Platform.

## Features

- PDF invoice generation with customizable templates
- Quote/estimate document creation
- Multi-currency support
- Company logo and branding
- REST API interface
- Ant Design-based admin panel

## Usage

```bash
# Part of the full platform
docker compose up -d
# API available at http://localhost:3002
```

## Architecture

Built with Node.js (Express) + React (Ant Design). Generates PDF documents via REST API calls.
