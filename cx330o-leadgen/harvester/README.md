# cx330o OSINT Engine

Open-source intelligence gathering module for the cx330o Sales Platform.

## Features

- Multi-source email and subdomain discovery
- Domain intelligence gathering
- DNS enumeration
- Search engine scraping
- API integrations (Shodan, VirusTotal, etc.)
- Multiple output formats (JSON, XML, HTML)

## Quick Start

```bash
pip install -r requirements.txt
python -m harvester -d example.com -b all
```

## Supported Sources

- Search engines (Google, Bing, DuckDuckGo, etc.)
- DNS brute force
- Certificate transparency logs
- Shodan, VirusTotal, SecurityTrails
- And more...
