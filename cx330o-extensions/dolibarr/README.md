# cx330o ERP Extension

Enterprise resource planning module for the cx330o Sales Platform.

## Features

- Contact and customer management
- Quotes, invoices, and orders
- Stock and inventory management
- Agenda and task management
- Human resources module
- ECM (Electronic Content Management)
- Manufacturing management
- Multi-company support

## Requirements

- PHP >= 7.2
- MariaDB, MySQL, or PostgreSQL
- Apache or Nginx web server

## Installation

See the `htdocs/` directory for the web application.
Follow the standard installation procedures in the documentation.
